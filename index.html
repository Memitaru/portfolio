Collaborated with six other remote developers to create and deploy this fullstack web app.

We used the Google Maps Places API to find relevant businesses and hours. A user can search by address to find nearby places that may have free wifi. A registered user can leave a review of a location they are working. If they are currently at the location they can also check the wifi speeds with the goal of showing average wifi speeds at each location once enough reviews are left.

I acted as team lead and contributer to Student ArtCo managing a team made up of six web developers, two iOS developers, two Android developers, and a UX designer.

Schools or home school groups can list art in order to get donations to fund their art programs. We used Cloudinary to store art images and SendGrid to allow potential buyers to contact schools. We were able to send this out to real users for feedback and a small homeschool coilition was able to raise $500 towards art supplies after including the link in their email newsletter!

I was team lead and contributer to this project leading a team of 4 other developers. We were working along side two other teams to continue adding features to this product.

Mission Control is a program management tool. We worked closely with stakeholders, program managers, and product managers to help guide the product direction.

My team's biggest contribution is the project list view. Here a program manager can customize the view to track the information most important to them. Each "status column" has a set of options set by the user that will appear in the drop down.

This product was my first time using many of these technologies. In eight weeks we were able to learn the tech stack and add our own feature that has become one of the core product features.
